# 🚲 Bike Resale Market Analysis Dashboard

A complete end-to-end Data Analytics project that collects used bike listings through **Web Scraping**, performs **Exploratory Data Analysis (EDA)** using Python, and builds an interactive **Power BI Dashboard** to analyze resale market trends.

---

## 📊 Dashboard Preview

![Dashboard](Bike%20Resale%20Price%20Dashboard.png)

---

# 📌 Project Overview

The objective of this project is to analyze the Indian used bike market by scraping real-world data from BikeKharido, cleaning and transforming the dataset, performing exploratory data analysis, and developing an interactive dashboard that helps identify factors affecting resale prices.

---

# 🎯 Objectives

- Scrape used bike listings from BikeKharido
- Clean and preprocess raw data
- Perform Exploratory Data Analysis (EDA)
- Identify factors affecting resale prices
- Build an interactive Power BI dashboard
- Generate business insights for buyers and sellers

---

# 🔍 Key Business Insights

- First-owner bikes dominate the resale market.
- Premium brands command significantly higher resale prices.
- Newer bikes generally retain higher resale value.
- Higher mileage negatively impacts resale prices.
- Metro cities contribute the largest share of bike listings.
- Ownership history plays an important role in determining resale value.
